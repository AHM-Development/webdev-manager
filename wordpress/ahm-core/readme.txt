Securely connects WordPress websites to AHM Webdev Manager.

== Installation ==

1. Upload and activate AHM Core.
2. In AHM Webdev Manager, create a pairing code for the website.
3. In WordPress, open Settings > AHM Core.
4. Enter the public AHM API URL and pairing code.

AHM Core never sends passwords or password hashes.
