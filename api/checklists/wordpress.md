# AHM WordPress Checklist

Version: 1.0.0

- WordPress core, PHP, active theme, and plugin versions are supported and updated.
- Required plugins configured in the website profile are installed and active.
- Inactive, abandoned, duplicated, or unknown plugins are reviewed.
- Administrator accounts are expected, current, and use least privilege.
- Password age is recorded from AHM Core events; passwords and password hashes are never collected.
- Debug display is disabled on production, file editing is disabled where policy requires it, and WordPress cron is healthy.
- Backups, caching, SMTP/email delivery, security monitoring, and image optimization are configured according to project policy.
- AHM Core version, connection status, heartbeat, and capabilities are current.
